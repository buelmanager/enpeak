# 앱 이름 변경 리서치

## 현재 이름 분석: EnPeak

| 항목 | 평가 |
|------|------|
| 의미 | "English" + "Peak" = 영어의 정점 |
| 글자수 | 6자 (2음절) |
| 한국어 발음 | 엔피크 (3음절) |
| 강점 | 짧고, 코인드 워드라 도메인 확보 용이, 성장/목표 지향적 |
| 약점 | "말하기/대화" 기능이 이름에서 드러나지 않음 |

- 트렌드: 게이미피케이션 이름 -> **액션/결과 지향적** 이름으로 이동 중

## 최종 추천 TOP 5

종합 평가를 바탕으로 가장 추천하는 이름 5개:

### 1. Talki (토키)
- **발음**: 한/영 모두 자연스러움 (2음절)
- **의미**: Talk + i = "나의 대화 파트너"
- **강점**: 핵심 기능(대화)을 직관적으로 전달, 친근하고 기억하기 쉬움
- **앱스토어**: `Talki - AI 영어회화 연습`
- **타겟**: 전 연령대

### 2. Convo (컨보)
- **발음**: 깔끔하고 모던 (2음절)
- **의미**: Conversation의 자연스러운 약어
- **강점**: 영어권에서 실제 사용하는 구어체, 세련된 느낌
- **앱스토어**: `Convo - AI 영어 대화 연습`
- **타겟**: 20-30대

### 3. Bloom (블룸)
- **발음**: 부드럽고 긍정적 (1음절)
- **의미**: 꽃피다, 성장하다
- **강점**: 성장 메타포가 학습과 완벽하게 맞음, 감성적
- **앱스토어**: `Bloom - AI 영어회화`
- **타겟**: 자기계발 관심층

### 4. Voxie (복시)
- **발음**: 독특하고 기억에 남음 (2음절)
- **의미**: Vox(목소리) + ie = "나의 목소리"
- **강점**: 음성/발음 학습 강조, 테크 느낌, 고유한 이름
- **앱스토어**: `Voxie - AI English Speaking`
- **타겟**: 테크 친화적 사용자

### 5. Lumi (루미)
- **발음**: 부드럽고 친근 (2음절)
- **의미**: Luminous(빛나는) = 지식의 빛
- **강점**: 범용성이 높고, 다국어 확장에도 어울림, 캐릭터화 용이
- **앱스토어**: `Lumi - AI 영어 학습`
- **타겟**: 초보자, 10-20대
