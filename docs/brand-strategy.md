## 3. 틈새시장 (Gap Analysis)

### 3-1. 발견된 시장 공백

| 공백 | 설명 | 기회 수준 |
|------|------|-----------|
| **가격대 공백** | 3,000~7,000원/월 AI 회화앱 부재 | 매우 높음 |
| **발음+회화 통합** | Speak=회화만, ELSA=발음만. 둘 다 하는 앱 없음 | 높음 |
| **공공장소 사용** | 지하철/카페에서 음성 없이 학습 불가 불만 | 높음 |
| **정직한 AI 평가** | "동문서답해도 점수 높게 줌" 불만 많음 | 중간 |
| **복습 시스템** | Speak 스마트 리뷰 불만 폭주, 제대로 된 SRS 부재 | 높음 |
| **문법+회화 통합** | 회화앱은 문법 약하고, 문법앱은 말하기 약함 | 중간 |

### 3-2. 사용자 핵심 불만 (경쟁사 리뷰 기반)

1. **"1주일이면 다 까먹는다"** -- 복습 시스템 부실 (Speak)
2. **"지하철에서 못 쓴다"** -- 공공장소 사용 불가 (전 앱)
3. **"발음 교정이 없다"** -- 회화앱의 구조적 한계 (Speak)
4. **"동문서답해도 점수가 높다"** -- AI 평가 부정확 (전 AI 앱)
5. **"5분마다 광고"** -- 무료 티어 사용성 최악 (Duolingo, Cake)
6. **"자동결제 덫"** -- 7일 체험 후 몰래 결제 (ELSA, TalkPal)

### 3-3. 우리 앱의 기존 강점

| 강점 | 설명 |
|------|------|
| 3모드 통합 | 자유대화 + 표현연습 + 롤플레이를 하나의 페이지에서 |
| 112개 시나리오 | 경쟁사 대비 압도적 롤플레이 콘텐츠 |
| RAG 기반 단어 확장 | 단어 -> 숙어 -> 예문 -> 대화로 자연스럽게 연결 |
| A1~C2 레벨 체계 | 2,661개 단어 + 5,375개 RAG 청크 |
| 텍스트+음성 동시 지원 | 공공장소 문제 해결 |
| PWA (앱스토어 불필요) | 즉시 접근, 설치 부담 없음 |

## 4. 브랜드 전략 제안: 3가지 타입

---

### TYPE A: "Convo" (컨보)

#### 컨셉 키워드
**"대화가 실력이 되는 순간"**

#### 브랜드 철학
영어 학습의 본질은 대화(Conversation)다.
문법 암기도, 단어 외우기도 결국 "대화"를 위한 것.
Convo는 대화 그 자체에 집중하는 앱이다.

#### 이름 분석

| 항목 | 내용 |
|------|------|
| 이름 | **Convo** |
| 발음 | 컨보 (2음절) |
| 의미 | Conversation의 구어체 약어. 영어권에서 실제 사용하는 단어 |
| 앱스토어 표기 | `Convo - AI 영어회화 연습` |
| 한글 표기 | 컨보 (Convo) |
| 태그라인 | "Your daily English conversation" |
| 도메인 | convo.app / getconvo.com / convo.ai |

#### 타겟

| 구분 | 내용 |
|------|------|
| 핵심 타겟 | **20~30대 직장인/취준생** |
| 확장 타겟 | 영어 회화에 관심 있는 전 연령대 |
| 페르소나 | "영어 문법은 알지만 실전에서 입이 안 떨어지는 사람" |
| 사용 시점 | 출퇴근길, 점심시간, 취침 전 10분 |

#### 핵심 기능 (차별화 포인트)

| 순위 | 기능 | 차별점 |
|------|------|--------|
| 1 | **3-in-1 Talk** | 자유대화/표현연습/롤플레이를 하나의 흐름에서 |
| 2 | **Smart Review** | 대화 중 틀린 표현 자동 수집 -> SRS 복습 카드 |
| 3 | **Silent Mode** | 텍스트 입력으로도 AI와 완전한 회화 연습 (공공장소 대응) |
| 4 | **112 Real Scenarios** | 카페, 병원, 면접 등 실생활 시나리오 112개 |
| 5 | **Daily Challenge** | 매일 달성 가능한 소규모 목표로 습관 형성 |

#### 디자인 컨셉

**스타일: Clean Conversational**
- Speak의 미니멀함 + Duolingo의 접근성을 결합
- 대화 인터페이스가 앱의 중심
- 깔끔하지만 차갑지 않은, "친구와 카톡하듯" 편안한 느낌

**메인 컬러**

|------|------|-----|------|
| Primary | Teal | **#0D9488** | 신뢰+성장. 경쟁사 미사용 영역 |
| Secondary | Warm White | **#FAFAF8** | 배경. 따뜻한 오프화이트 |

### TYPE B: "Talki" (토키)

#### 컨셉 키워드
**"매일 10분, AI 친구와 영어 수다"**

#### 브랜드 철학
영어를 "공부"가 아닌 "수다"로 만든다.
부담 없이 매일 AI 친구와 가볍게 이야기하다 보면
어느새 영어가 입에서 나오는 경험.

#### 이름 분석

| 항목 | 내용 |
|------|------|
| 이름 | **Talki** |
| 발음 | 토키 (2음절) |
| 의미 | Talk + i (나의 토크 친구) |
| 앱스토어 표기 | `Talki - AI 영어 대화 친구` |
| 한글 표기 | 토키 (Talki) |
| 태그라인 | "Talk easy, learn naturally" |
| 도메인 | talki.app / gettalki.com / talki.ai |

#### 타겟

| 구분 | 내용 |
|------|------|
| 핵심 타겟 | **10~20대 영어 초보~중급자** |
| 확장 타겟 | 영어 공부를 가볍게 시작하고 싶은 모든 연령 |
| 페르소나 | "영어 학원은 무섭고, 혼자 공부는 지루한 사람" |
| 사용 시점 | 쉬는 시간, 자기 전, 심심할 때 5분 |

#### 핵심 기능 (차별화 포인트)

| 순위 | 기능 | 차별점 |
|------|------|--------|
| 1 | **AI Buddy** | 캐릭터화된 AI 친구 (이름, 성격 부여) |
| 2 | **Bite-size Roleplay** | 3분짜리 미니 시나리오 (부담 없이 시작) |
| 3 | **Expression Collection** | 배운 표현 스티커처럼 수집하는 컬렉션 |
| 4 | **Daily Streak Challenge** | 연속 학습 + 리워드 (게이미피케이션) |
| 5 | **Quick Quiz** | 홈 화면에서 바로 풀 수 있는 1문제 퀴즈 |

#### 디자인 컨셉

**스타일: Friendly & Playful**
- Duolingo의 친근함 + Cake의 밝은 에너지를 결합
- 둥글둥글한 UI 요소, 부드러운 곡선
- 캐릭터 일러스트 활용 가능
- "학습"보다는 "놀이" 느낌

## 6. 추천 의견

### 시장 기회 기준 추천: **TYPE A - Convo**

**이유:**

1. **가장 넓은 타겟**: 20~30대는 영어학습앱 사용자의 45.3%를 차지
2. **명확한 포지셔닝**: "AI 영어 대화앱" 카테고리에서 Speak 다음 포지션
3. **Teal 컬러**: 경쟁사 중 미사용 영역이라 시각적 차별화 극대화
4. **이름의 범용성**: "Convo"는 영어/한국어 모두에서 자연스럽고 확장성 좋음
5. **가격 경쟁력**: Speak보다 저렴하면서 핵심 기능 모두 제공
6. **기술 스택과의 정합성**: 현재 3모드 통합 아키텍처가 Convo 컨셉과 완벽하게 부합

### 만약 차별화 극대화를 원한다면: **TYPE B - Talki**
- Duolingo 사용자 중 "게임은 재밌는데 실제로 말을 못하겠다"는 불만이 많음
- Talki는 그 사용자를 "가볍게 말하기 연습"으로 전환시키는 포지셔닝

### 만약 프리미엄 시장을 노린다면: **TYPE C - Verba**
- Ringle(14만원+)은 너무 비싸고, Speak은 피드백이 부정확하다는 틈새
- "결과를 내는 영어 훈련"이라는 메시지로 B2B 확장 가능성도 있음
